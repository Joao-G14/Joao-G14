## 🎯 Objetivo Profissional

> *"Os dados do presente são o mapa para o futuro."*

Meu objetivo é me **profissionalizar completamente em Análise de Dados**, construindo uma base técnica sólida que me permita evoluir naturalmente para **Cientista de Dados**. Por isso, invisto meu tempo de estudo no aprendizado de **Machine Learning** — desde regressões e classificações até redes neurais com TensorFlow.

Acredito que dominar ML dentro da análise de dados é o diferencial que transforma insights em **previsões reais**.
